Se necesita usar los atributos "action" (donde se agrega un URL de donde se recive información) y "method" (que tiene los métodos "get" y "post")

Se utiliza para crear una tabla, se usan los tags "tr" para hacer las filas de la tabla y dentro de este el tag "td" para añadir una celda a la fila con su respectiva información.
